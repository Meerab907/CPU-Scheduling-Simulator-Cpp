CPU Scheduling Simulator (C++)

Overview
This project is an educational CPU Scheduling Simulator developed in C++. It simulates major CPU scheduling algorithms and visualizes their execution using colored Gantt charts in the terminal. The simulator calculates key performance metrics like Waiting Time (WT), Turnaround Time (TAT), and Response Time (RT), providing an interactive way for students to understand scheduling concepts.

Features

Simulation of key CPU scheduling algorithms:

First Come First Serve (FCFS)

Shortest Job First (SJF - Non-preemptive)

Shortest Remaining Time First (SRTF - Preemptive)

Priority Scheduling (Non-preemptive)

Round Robin Scheduling

User-defined input:

Number of processes

Arrival time

Burst time

Priority

Time quantum (for Round Robin)

Interactive menu-based console program

Visual output: Colored Gantt charts for process execution timelines

Calculation and display of WT, TAT, and RT for all processes
